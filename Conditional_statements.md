# conditional statements are used to run certain part of code according to the output of condition

```c
// if( condition ){
// executes if conditon is true
// always put condition in (condition) bracket
// }
// else{
// executes if condition is false
// }

// example
int a = 4;

if(a<5)
{
  printf("yes");
  
}
else {
printf("no");
}
```
