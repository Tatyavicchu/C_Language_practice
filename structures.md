# STRUCTURES
 `A collection of values of diffrent data types.`
 `structure is an user defined data types which can hold multiple types of data type.`
 ### `define`
 ```c
 struct students s1;
 s1.cgpa=7.5;
 s1.name=mohit;
 s1.roll=39;
 ```
 ### `Syntax`
 ```c
 struct student{
 char name[100];
 int roll;
 float cgpa;
 } ; // statement terminator(;) is necessary.
 ```
